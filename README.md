# TaskFlow

## 🚀 Features

### 🔐 Authentication & Security

* User registration and login functionality
* Secure password handling
* Protected routes for authorized users

---

### 📝 Task Management

* Create, update, and delete tasks
* View all tasks in an organized dashboard
* Set deadlines for tasks
* Edit task details anytime

---

### 📊 Priority Management (Quadrant-Based) ⭐

* Tasks are categorized using a **quadrant system**:

  * Important & Urgent
  * Important but Not Urgent
  * Not Important but Urgent
  * Low Priority
* Helps users focus on what truly matters
* Improves productivity and time management

---

### 🎯 User Experience

* Clean and responsive UI
* Easy navigation and task tracking
* Structured layout for better clarity

---

### 🔗 Backend & API

* RESTful API integration
* Built with Express.js and MongoDB
* Efficient data handling using Mongoose

---

### ⚙️ Configuration

* Environment variables using `.env`
* Scalable and modular project structure

---

## ⭐ Advanced Highlights

* Implements **Eisenhower Matrix concept** for task prioritization
* Full-stack MERN application
* Designed with real-world productivity use cases
* Deployment-ready architecture
